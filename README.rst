Silex-Markdown
================

Installation
------------

Create a composer.json in your projects root-directory::

    {
        "require": {
            "mheap/Silex-Markdown": "*"
        }
    }

and run::

    curl -s http://getcomposer.org/installer | php
    php composer.phar install
